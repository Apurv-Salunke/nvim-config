# LazyVim

This is a minimal LazyVim configuration template.

## Features

- Catppuccin Mocha theme
- Python development setup (ruff, pyright)
- Snacks.nvim for picker and explorer

## Installation

```bash
git clone https://github.com/LazyVim/LazyVim.git ~/.config/nvim
rm -rf ~/.config/nvim/.git
```

## Customization

Add your custom plugins in `lua/plugins/`
